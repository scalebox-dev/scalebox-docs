## Local Preview

Install Mintlify CLI:

```bash
npm i -g mintlify

```

Run in the docs directory:

```bash
mintlify dev

```

Then visit `http://localhost:3000` to view the documentation.

## Build Documentation

```bash
mintlify build

```
