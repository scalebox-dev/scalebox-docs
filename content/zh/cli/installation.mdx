# ScaleBox CLI 安装指南

下载并安装 ScaleBox CLI 工具，从命令行管理模板和沙盒。
支持 Linux、macOS 和 Windows 平台，提供原生二进制文件，无需额外依赖。

> ✨ 最新版本：ScaleBox CLI v1.2.6 - 模板名称预检查

## 平台下载

选择适合您平台的二进制文件。所有二进制文件都是静态编译的，无需额外依赖。
